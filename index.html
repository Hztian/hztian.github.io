HTML常用标签整理

HTML常用标签整理

标签 描述
<a> 标签定义超链接，用于从一张页面链接到另一张页面。
<audio> 标签定义声音，比如音乐或其他音频流。
<body> 标签定义文档的主体
<br> 可插入一个简单的换行符。
<buttom> 标签定义一个按钮 , 在 button 元素内部，您可以放置内容，比如文本或图像。
<canvas> 标签定义图形，比如图表和其他图像。
<em> 把文本定义为强调的内容。
<strong> 把文本定义为语气更强的强调的内容。
<code> 定义计算机代码文本。
<kdb> 定义键盘文本。它表示文本是从键盘上键入的。它经常用在与计算机相关的文档或手册中。
<cite> 定义引用。可使用该标签对参考文献的引用进行定义，比如书籍或杂志的标题。
<dl> 标签定义了定义列表（definition list）。
<dt> 标签定义了定义列表中的项目（即术语部分）。
<dd> 在定义列表中定义条目的定义部分。
<dialog> 标签定义对话框或窗口。
<dir> 标签定义目录列表。
<div> 可定义文档中的分区或节（division/section）。
<fieldset> 标签将表单内容的一部分打包，生成一组相关表单的字段。
<figure> 标签规定独立的流内容（图像、图表、照片、代码等等）。
<footer> 标签定义文档或节的页脚。
<form> 标签用于为用户输入创建 HTML 表单。
<h1>-<h6> 标签可定义标题。
<header> 标签定义文档的页眉（介绍信息）。
<hr> 标签在 HTML 页面中创建一条水平线。
<i> 标签显示斜体文本效果。
<img> 元素向网页中嵌入一幅图像
<input> 标签用于搜集用户信息。
<label> 标签为 input 元素定义标注（标记）。
<li> 标签定义列表项目。 可用在有序列表 (<ol>) 和无序列表 (<ul>) 中 。
<link> 标签定义文档与外部资源的关系。
<main> 标签规定文档的主要内容。
<mark> 标签定义带有记号的文本。请在需要突出显示文本时使用 <m> 标签。
<menu> 标签定义命令的列表或菜单。
<meter> 标签定义已知范围或分数值内的标量测量。也被称为gauge（尺度）。
<nav> 标签定义导航链接的部分
<ol> 标签定义有序列表。
<p> 标签定义段落。
<section> 标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分。
<select> 元素可创建单选或多选菜单， 元素中的<option>标签用于定义列表中的可用选项。
<source> 标签为媒介元素（比如 <video>和<audio>）定义媒介资源。
<span> 标签被用来组合文档中的行内元素。
<sub>,<sup> 标签可定义上下标文本。
<table> 标签定义 HTML 表格。
<tr> 标签定义 HTML 表格中的行。
<td> 标签定义 HTML 表格中的标准单元格。
<th> 定义表格内的表头单元格。
<u> 标签可定义下划线文本。
<ul> 标签定义无序列表。
<video> 标签定义视频，比如电影片段或其他视频流。

发表于 2019-05-14 | 更新于 2019-06-15 | 分类于 HTML

JavaScript的七种数据类型

  1. number(数字)

    十进制：

    1

    1.1 可以用.1表示0.1

    1.11e2 表示1.11*10²

二进制：

0b开头 如：0b11等于十进制的3

八进制：

如：011

十进制：

如：0x11

  1. String(字符串)

    ‘’ 和 “” 单引号或双引号里面是内容

    要注意的是，如果你定义的字符串内容里有‘ 和“还有\（单引号和双引号还有\）,则需要\（反斜杠，也叫转义符）来转义。

  1. boolean(布尔)

    就是ture 和 false。

    a&&b(a与b):

    | a/b | 真 | 假 |
    | —- | —- | —- |
    | 真 | √ | × |
    | 假 | × | × |

    只有a和b都为真时，a&&b才为真。

a||b(a或b)

a/b 真 假
真 √ √
假 √ ×

只要a或b有一个为真，a||b就为真。

  1. Symbol(字符)

  1. null

    只有一个值，就是null，表示什么也没有

  1. undefined

只有一个值，就是undefined，表示什么也没有

只有JS有两个是表示空，算是JS设计的一个bug吧

null和undefined的区别：

①变量没有值 ——JS推荐使用undefined

②有一个对象（object），现在不想赋值——推荐使用null

​ 有一个非对象，现在不想赋值——推荐使用undefined

  1. object(对象)

    哈希表也是对象，对象里可以还有对象，对象就是简单类型的组合。
